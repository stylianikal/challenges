This repository contains the source code for the developer challenges.

A copy of this repository is made available to candidates. Note that
this is only a copy; commits that candidates while answering questions
are not propagated back to the master repository.

If you are an Albourne employee, please ensure that all commits to
this repository use the following git identity:

    git config user.name "Albourne Developers"
    git config user.email "dummy@example.com"

If you are a candidate, please configure your git identity to use the
name and email address on your CV:

    git config user.name "Put Your Name Here"
    git config user.email "your@email.address"
